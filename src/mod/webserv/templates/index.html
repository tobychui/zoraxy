Welcome to Zoraxy Static Web Server!

Your static web server is up and running

If you see this page, that means your static web server is running. By default, all the html files are stored under ./web/html/ relative to the Zoraxy runtime directory.

To manage your website files and customize error pages, use the WebDAV File Server feature in Zoraxy. You can see more details in the Static Web Server > WebDAV Server section of the Zoraxy admin panel.

For online documentation, please refer to zoraxy.aroz.org or the project wiki.

Thank you for using Zoraxy!
