Configuration Error

Zoraxy Auth is not properly configured

Error: The SSO Redirect URL is not configured. This setting is required for Zoraxy Auth to function properly.

How to Fix This

  1. Log in to your Zoraxy admin panel
  2. Navigate to SSO Settings → Zoraxy Auth tab
  3. Configure the SSO Redirect URL field with your authentication server URL
  4. Save the settings and try again

Tip: The SSO Redirect URL should point to the Zoraxy Auth gateway endpoint where users will log in. Make sure you have added a HTTP proxy rule to forward the SSO domain to the Zoraxy Auth SSO port.
