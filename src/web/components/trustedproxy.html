Trusted Proxy

Configure trusted upstream proxy IP addresses or CIDR ranges.
When a request arrives from a trusted proxy, Zoraxy will read the real client IP from forwarded headers (e.g. X-Forwarded-For or X-Real-IP) instead of the connection address. Only add proxies you control.
